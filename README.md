A simple icon library

## Usage

To install the package use the following command:

```bash
pnpm add @tbdc-agro-softwares/icons
```

Usage with Vue 3:

```html
<!-- Example.vue -->
<script>
import AddCircleOutline from '@tbdc-agro-softwares/icons/AddCircleOutline.svg'
</script>

<template>
    <AddCircleOutline />
</template>
```
